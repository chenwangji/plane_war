# plane_war

A small game of plane war base on javasccript prototype and extend.

### 玩法

1. 选择关卡。

	总共有4个关卡，由易到难，且只有一条命。

2. 进入游戏。

	通过鼠标拖动己方飞机摧毁和躲避敌机。


### 代码思路

1. 结构
2. 开始界面样式
3. 封装导演类，将所有的主逻辑放到该类的属性和方法中，通过事件队列，计时器事件完成后callBack依次执行
4. 封装飞机，子弹的父类原型
5. 继承，并且扩展我方飞机，子弹，敌机的属性和方法
